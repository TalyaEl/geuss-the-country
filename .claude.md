AI Development Rules for "Guess the Country"
This project follows specific engineering constraints to ensure high performance and maintainability.
Core Rules:
1. Asynchronous Flow: All database and API operations must use async/await.
2. Stateless Backend: No server-side sessions. All state (like country_id) must be managed by the frontend and passed in API requests.
3. Minimal UI: Follow a "Functionality-First" design. Avoid external CSS/JS frameworks.
4. Validation: Country name matching must be case-insensitive and trim whitespace.

Tech Stack Constraints
* Backend: FastAPI
* Database: SQLite with SQLAlchemy (Async)
* Frontend: Vanilla HTML5, CSS3, and JavaScript
